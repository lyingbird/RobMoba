### 🧠 MOBA 设计知识库

**王者荣耀轻量化设计精髓**:
- **3+1技能体系**: 比LOL少一个技能，但每个技能承担更多功能（如宫本一技能：位移+伤害+标记）
- **5分钟节奏控制**: 经济膨胀快→装备成型早→团战提前→单局10-15分钟
- **英雄定位清晰**: 战/法/射/辅/刺，每个定位有明确的对局角色
- **被动技能设计**: 定义英雄独特性（如后羿被动：普攻叠加攻速，鼓励持续输出）
- **技能连招深度**: 简单技能组合产生丰富策略（如安琪拉213连招、廉颇1231连招）

**LOL 核心机制参考**:
- **对线博弈**: 补刀/消耗/换血 → 轻量化为"区域控制+经验争夺"
- **视野系统**: 草丛/守卫 → 轻量化为"关键区域视野规则"
- **团战位置**: ADC站位/刺客切入/坦克开团 → 保留但简化地图规模
- **拉克丝技能设计**: Q根+W护盾+E减速区+R直线，技能互相联动的典范

**Roblox MOBA 适配策略**:
- **单局时长**: 3-8分钟（当前3杀胜利 → 可考虑水晶/基地模式）
- **英雄数量**: 精不在多，8-12个英雄就能形成有趣的克制网（当前4个，目标扩到8个）
- **经济系统**: 简化为"击杀=经验+金币 → 买装备/升级"，去掉补刀
- **地图规模**: 1v1/2v2/3v3为主，不做5v5大地图（Roblox性能&匹配考虑）
- **社交设计**: 快速匹配 + 好友组队 + 击杀播报 + 赛后数据 + MVP展示

**当前项目英雄池理解**:
- **拉克丝** (1002-1005): 法师/控制，Q根W盾E减速区(Recast)R直线轰炸
- **安琪拉** (1006-1008): 法师/爆发，Q散射W眩晕+漩涡 R引导鼠标追踪
- **后羿** (1009-1011): 射手/持续，Q多重箭矢 W持续AOE R远程眩晕
- **廉颇** (1012-1015): 坦克/突进，Q冲锋击飞 W减速+护盾 R三连击/电影特写
