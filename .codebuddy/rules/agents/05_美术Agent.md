### 🧠 Roblox UI 知识库

**项目 UI 架构理解**:
```
UIManager (facade模式, UIManager.lua)
├── UI_HUD (857行) — 主界面：技能栏6槽(QWERDF)+HP/MP条+属性面板+等级经验+计分板+死亡遮罩+胜负弹窗
├── UI_Backpack (236行) — 背包：物品格子+毛玻璃背景+BlurEffect
├── UI_DragDrop (471行) — 拖拽：技能→技能槽、装备→装备槽、符文→技能符文槽、悬停Tooltip
└── UI_HeroSelect (358行) — 英雄选择：5张卡片+入场动画+yield等待
```

**UITheme 配色系统 (原神风)**:
- 暗色HUD: 半透明黑底 (`Color3.fromRGB(20,20,30)`, 0.7透明度)
- 技能栏: 亮色边框 + CD覆盖层（灰色→原色渐变）
- 血条红/蓝条蓝/经验条金 — 渐变效果
- 统一圆角: `UICorner.CornerRadius = UDim.new(0, 8)`
- 统一阴影/描边: `shadow()`/`stroke()` 工具函数

**MOBA UI 设计规范**:
- 技能栏: 底部居中，6个正方形槽位，CD遮罩从上到下扫过
- HP/MP条: 玩家信息区左上角，HP红色/MP蓝色
- 头顶UI (BillboardGui): 敌方红色/友方绿色血条 + 等级标识
- 击杀播报: 屏幕上方居中，3秒淡出
- 死亡遮罩: 全屏半透明黑+倒计时大字
- 小地图(待实现): 右下角，显示队友/敌人(被发现时)/野怪位置

**Roblox 特有 UI 元素**:
- **ViewportFrame**: 3D物体在UI中渲染（技能指示器用此实现3D地面投影）
- **BillboardGui**: 跟随3D物体的UI（头顶血条/伤害数字）
- **UIAspectRatioConstraint**: 跨设备保持比例
- **AutomaticSize**: 文本自适应大小
